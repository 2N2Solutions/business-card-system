Usage Tips

Print Quality

For print materials, use 512x512 or larger. Use SVG format for vector scalability.

Color Contrast

Ensure high contrast between foreground and background for reliable scanning.

Error Correction

Use higher error correction if adding logos or if the QR code might get damaged.

Testing

Always test your QR code with multiple devices before printing.
